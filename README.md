HR Data Analytics Dashboard (Power BI)

 Overview

This project analyzes HR data to understand employee attrition, performance, and workforce trends using Power BI.

 Key Features

* Attrition analysis
* Department-wise insights
* Salary & job role analysis
* Interactive dashboard

 Tools Used

* Power BI
* Excel / CSV

 Files

* HR Analytics `.pbix` file
* Dataset
* Dashboard screenshot

 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop

Insights

* Attrition varies across departments
* Salary and job satisfaction impact retention
